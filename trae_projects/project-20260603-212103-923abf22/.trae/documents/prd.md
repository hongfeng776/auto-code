## 1. 产品概述

库存补货协同看板是一款面向供应链运营团队的实时监控与决策支持工具，旨在将仓库库存状态、SKU 风险识别、补货建议生成、供应商协同状态和趋势分析整合在统一视图中，帮助运营人员快速发现断货风险、追踪补货进度、协同供应商响应，并通过本地模拟预警机制提前预判库存异常。

- 目标用户：供应链运营经理、仓库管理员、采购专员
- 核心价值：降低断货率、缩短补货响应周期、提升供应商协同效率

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 运营经理 | 管理员分配 | 查看全量看板、审核补货建议、触发预警模拟 |
| 仓库管理员 | 管理员分配 | 查看所属仓库数据、确认入库、标记异常 |
| 采购专员 | 管理员分配 | 查看补货建议、联系供应商、更新采购状态 |

### 2.2 功能模块

1. **看板主页**：仓库概览、SKU风险摘要、预警通知、趋势迷你图
2. **SKU风险详情**：风险分级列表、异常处理逻辑展示、关联补货建议
3. **补货建议面板**：建议列表、供应商状态联动、一键生成采购单
4. **供应商协同视图**：供应商状态矩阵、响应时效追踪、缺失供应商标记
5. **趋势概览页**：库存趋势图、预警频率趋势、补货效率指标
6. **模拟预警中心**：本地参数调节、预警模拟运行、结果展示与处理建议

| 页面名称 | 模块名称 | 功能描述 |

## 3. 核心流程

### 3.1 库存风险识别与补货流程

用户进入看板 → 查看仓库概览发现异常仓库 → 点击进入SKU风险详情 → 识别高风险/断货SKU → 查看关联补货建议 → 检查供应商状态 → 生成采购单/标记寻源 → 跟踪补货进度

```mermaid
flowchart TD
    "进入看板主页" --> "查看仓库概览"
    "查看仓库概览" --> "发现异常仓库"
    "发现异常仓库" --> "进入SKU风险详情"
    "进入SKU风险详情" --> "库存=0?"
    "库存=0?" -->|"是"| "标记紧急断货 - 启动紧急补货流程"
    "库存=0?" -->|"否"| "库存低于安全线?"
    "库存低于安全线?" -->|"是"| "生成补货建议"
    "库存低于安全线?" -->|"否"| "标记为正常"
    "生成补货建议" --> "有可用供应商?"
    "有可用供应商?" -->|"是"| "关联供应商 - 生成采购单"
    "有可用供应商?" -->|"否"| "标记无供应商 - 触发寻源流程"
    "关联供应商 - 生成采购单" --> "跟踪补货进度"
    "标记无供应商 - 触发寻源流程" --> "跟踪补货进度"
```

### 3.2 模拟预警流程

用户进入模拟预警中心 → 调节参数 → 运行模拟 → 查看预警结果 → 预警过多时触发批量处理建议 → 确认处理方案

```mermaid

    "提示批量处理方案" --> "确认处理方案"

```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深炭灰 (#1a1a2e) 背景 + 琥珀橙 (#f59e0b) 强调色 + 翡翠绿 (#10b981) 安全色 + 玫红 (#ef4444) 危险色
- **按钮风格**：圆角微凸（rounded-lg + shadow-sm），主操作用琥珀橙，危险操作用玫红
- **字体**：标题用 "DM Sans"（几何感现代字体），正文用 "Noto Sans SC"（中文优化）
- **布局风格**：卡片式布局 + 左侧导航栏，顶部预警横幅
- **图标**：Lucide 图标库，线性风格
- **氛围**：工业数据仪表盘风格，深色主题，微妙的网格背景纹理

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |

### 4.3 响应式设计

- 桌面优先设计，最小宽度 1280px
- 1440px+ 时看板展开为四列布局
- 1280-1440px 时自动折叠为双列
- 不支持移动端（企业级看板场景以桌面为主）
